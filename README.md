## Overview

**Accel RFP** is a **unified, map-centric platform** for:

- **Poster/Evaluator Mode**: Publish RFPs with geographic radius, custom fields, and scoring weights  
- **Searcher/Responder Mode**: Browse RFPs on map, filter by location/industry, submit responses  
- **Automated Scoring & Ranking**: Responses auto-evaluated against preset thresholds  
- **Admin Dashboard**: Full CRUD for templates, pricing, config, and response review  

Built with:
- **Frontend**: HTML, CSS, JS, Leaflet
- **Backend**: Google Apps Script + Google Sheets
- **No server. No database. No cost.**
